JSON
    = JSON 데이터를 Input Data 입력란에 입력 후 Run, Ctrl + Enter
      + Focus Data: 강조 표시할 대상 입력후 Run, Ctrl + Enter (대소문자를 구분하지 않음, Enter로 구분하여 복수선택 가능)
        . # : 표현식 주석 처리 (표현식의 시작에 사용)
        . >A : 키가 A인 요소 선택 (표현식의 시작에 사용)
        . A* : A로 시작하는 요소 선택
        . *A : A로 끝나는 요소 선택
        . A / B : A 한단계 아래 B 선택
        . A / 0 : A 한단계 아래 배열의 첫번째 요소 선택
        . A / * : A 한단계 아래의 자식요소 모두 선택
        . A / ** : A 아래 모든 계층구조의 자식요소 선택 (A를 포함하지 않음, 표현식의 끝에 사용)
        . A** : A 아래 모든 계층구조의 자식요소 선택 (A를 포함, 표현식의 끝에 사용)
      + View Data: 화면에 제한적으로 출력할 대상 입력후 Run, Ctrl + Enter (대소문자를 구분하지 않음, Enter로 구분하여 복수선택 가능)
        . Focus Data와 동일한 표현식 사용
        . 복수 선택일 경우 구조 해체후 배열로 표시
      + [A]: 배열 생략 - 활성화 시 보여지는 입력란의 숫자만큼만 배열 내용 표시
      + [O]: 객체 생략 - 활성화 시 보여지는 입력란의 숫자만큼만 객체 내용 표시
      + [R]: 키 검색에 정규표현식 사용 (>A, A*등 표현식의 고유 문법은 유지)

  MindMap, Tree
    = 엑셀 시트 및 단계별 Tab으로 구분되는 데이터를 복사하여 Input Data 입력란에 붙여넣은 후 Run, Ctrl + Enter
      + [S]: 노드에 그림자 표시
      + [D]: 노드의 깊이를 [Depth]로 표시
      + [E]: 비어있는 [Depth] 제거
      + Focus: 강조 표시할 노드의 깊이

  공통
      + [T]: 검색란 투명 처리
      + [F]: 데이터 입력란 리사이징 유지
      + Show: 화면에 표시할 단계 선택
        . 선택된 값이 마지막 Depth(전체표시)가 아닐 경우 배경이 노란색
      + Zoom: 화면에 출력된 이미지 크기 조절
      + Save: 화면을 이미지로 저장
      + Share: 화면 공유용 URL을 생성 후 클립보드 저장
      + Copy: 화면의 데이터를 클립보드에 저장
      + Sample: 샘플 데이터를 화면에 표시

  테스트: 
    = Type 선택 후 Sample 버튼 클릭
